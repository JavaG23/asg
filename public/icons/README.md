# App Icons

To create proper app icons for ASG App:

## Quick Method (Online Tool):
1. Go to https://realfavicongenerator.net/ or https://www.pwabuilder.com/imageGenerator
2. Upload your logo/icon (ideally 512x512px PNG with transparent background)
3. Download the generated icon pack
4. Replace the placeholder icons in this directory

## Required Icons:
- `icon-192x192.png` - For mobile home screen
- `icon-512x512.png` - For app splash screen and high-res displays
